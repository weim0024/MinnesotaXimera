\begin{problem}

\begin{problem}

$y= \answer{x^3}$.

\begin{problem}
Write the operation of raising 10 to a power (e.g., when the input is 2, the output is $10^2$) as a function of $x$:

$y= \answer{10^x}$.

What operation undoes raising 10 to a power? Write this operation as a function of $x$.

$y=\answer{\log_{10}(x)}$
\end{problem}
